**!!! the program will freeze when you press a button !!!**

to run if you are on windows you can run the .exe file.

<details>
<summary>non windows OS</summary>
if you do not have windows make sure you have python3.12 or newer
make a python venv using the command

```powershell
python -m venv venv
```

install python_weather using pip:

```powershell
pip install python-weather
```

or

```powershell
python -m pip install python-weather
```

then open up the directory named code_tkinter and run the file main.py
</details>
